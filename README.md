# ₿ Cryptocurrency Analytics Dashboard

An interactive Power BI dashboard that provides comprehensive analysis of cryptocurrency market performance, price trends, trading volume, and comparative asset behavior.

This project helps analysts, investors, and researchers monitor crypto performance, identify trends, and evaluate market movements across multiple digital assets.

--
## 📌 Project Objective

The goal of this dashboard is to:

- Analyze cryptocurrency price trends over time

- Compare performance across multiple digital assets

- Identify best and worst performing cryptocurrencies

- Monitor trading volume fluctuations

- Evaluate long-term market behavior

- Provide actionable insights for investment analysis

## 📊 Dashboard Modules
### 🧭 1. Navigation Panel

The dashboard includes three main analytical sections:

- Dashboard Overview

- Pricing Analysis

- Overall Market Analysis

### 📈 2. Market Performance Dashboard
Key Metrics

- Final Close Price

- Highest Close Price

- Lowest Close Price

- Average Daily Volume

Visual Insights

- Price trend with high/low comparison

- 30-day moving average analysis

- Trading volume trends over time

- Multi-cryptocurrency selection filter

- Interactive date range filtering

This module provides a time-series view of market behavior and volatility.

### 💰 3. Pricing Analysis Module
Price Comparison Table

- Detailed comparison across cryptocurrencies including:

- Final Price

- 30-Day Percentage Change

- 365-Day Percentage Change

- 365-Day High

- Percentage Change from High

- Average Daily Volume

Performance Analysis

- Best performers (30-day & 365-day)

- Worst performers (30-day & 365-day)

- Relative growth comparison

This section enables quick performance benchmarking across assets.

### 🔬 4. Overall Cryptocurrency Analysis
Comparative Market Table

- Multi-asset price comparison by date

- Cross-cryptocurrency trend analysis

- Aggregate performance summary

Correlation Insights

- Price correlation ranking across cryptocurrencies

- Identifies strongly related market movements

### ✨ Key Features
